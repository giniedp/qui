# Qui
qui-lib is a lightweight ui controls library for javascript.

## Installation

### NPM

`npm -i qui-lib`

### CDN

```html
<style link="https://unpkg.com/qui-lib@x.x.x/dist/qui.min.css"></style>
<script src="https://unpkg.com/qui-lib@x.x.x/dist/qui.umd.min.js"></script>
```
